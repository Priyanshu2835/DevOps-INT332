1\. Container Lifecycle Management

Run a container (creates and starts it) 

docker run -d -p 80:80 --name my\_container nginx

Start a stopped container 

docker start <container\_id/name>

Stop a running container 

docker stop <container\_id/name>

Restart a container 

docker restart <container\_id/name>

Pause a container 

docker pause <container\_id/name>
