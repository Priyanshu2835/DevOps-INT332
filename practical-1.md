\# Practical 1: Docker Installation \& Basic Commands



\## Aim

To understand basic Docker commands.



\## Questions

1\. Check Docker version.

2\. Pull an Ubuntu image.

3\. Run a container in interactive mode.

4\. List running containers.

5\. Stop a container.



\## Commands



docker --version

docker pull ubuntu

docker run -it ubuntu

docker ps

docker stop <container\_id>
